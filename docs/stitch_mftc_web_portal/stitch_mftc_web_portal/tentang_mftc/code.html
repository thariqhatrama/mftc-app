Tentang MFTC

Muslim-friendly Tourism Certification (MFTC) hadir untuk mengangkat standar pariwisata Indonesia ke tingkat global. Kami berkomitmen untuk membangun ekosistem wisata yang inklusif, terpercaya, dan sepenuhnya memenuhi kaidah Halal.
