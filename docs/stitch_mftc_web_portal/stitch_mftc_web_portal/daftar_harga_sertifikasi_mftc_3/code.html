Pertanyaan Umum (FAQ)

Sertifikasi MFTC berlaku selama 2 (dua) tahun kalender. Re-audit disarankan dilakukan 3 bulan sebelum masa berlaku berakhir untuk memastikan kontinuitas status "Certified".
Tidak ada biaya maintenance tahunan. Investasi yang Anda bayarkan sudah mencakup seluruh proses audit dan penggunaan logo selama masa berlaku sertifikat.
Kami akan memberikan laporan detil "Corrective Action" (CAPA). Anda memiliki waktu 30 hari untuk melakukan perbaikan tanpa biaya audit tambahan untuk verifikasi kedua.
Perbedaan utama terletak pada kedalaman audit lapangan, intensitas pendampingan (coaching), serta dukungan pemasaran premium yang didapatkan setelah tersertifikasi.
